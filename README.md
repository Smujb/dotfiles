# SmuJB's Personal Dotfiles

These are the dotfile configurations I use in my current daily driver system, [Amethyris](https://github.com/Smujb/amethyris).

This includes my preferred window manager (sway) as well as my preferred terminal and shell (kitty and zsh) plus utilities for all 3.

Due to the base system being atomic, almost every other app is installed either as a flatpak or using homebrew - including zsh extensions.
